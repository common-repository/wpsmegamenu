    wpsmegamenu is Elemntor Addon and Theme Making Mega Menu with Short-code as Custom Post

== Description ==
    You can use this plugin to make any WordPress Mega Menu simple with ShortCode

== Installation ==
This section describes how to install the plugin and get it working Best Way.
e.g.
1. Upload `wpsmegamenu` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the elementor editor.

### Features

* Unlimited Eelmntor Widget Options
* WooCommerce Widgets
